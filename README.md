# Chile 2050: Nueva Arquitectura Económica

Este repositorio contiene el **Blueprint Geoeconómico y Laboratorio Algorítmico** para la transformación de Chile hacia el año 2050. Basado en los principios de la **Economía de la Complejidad**, el **Estado Emprendedor** y el **Enfoque de Capacidades Humanas**.

## 🚀 Propósito del Proyecto
Transformar el modelo extractivista chileno en una potencia geoeconómica de energía limpia, complejidad tecnológica y servicios avanzados en la cuenca del Indo-Pacífico, subordinando el crecimiento económico al desarrollo de las libertades y capacidades reales de sus ciudadanos.

## 🛠 Estructura de la Plataforma (React + Vite)
- **Informe Rector (13 Fases)**: Auditoría estratégica exhaustiva inspirada en estándares OCDE/McKinsey.
- **Laboratorio de Simulación (Fase 11)**: Modelos interactivos de impacto para Hidrógeno Verde y Litio (Cátodos LFP).
- **Benchmarks Internacionales**: Comparativa con los modelos de Singapur, Corea del Sur, Noruega y Estonia.
- **Blueprint 2050 (Fase 13)**: Diagrama metabólico interactivo (Mermaid.js) que desglosa el sistema nacional.
